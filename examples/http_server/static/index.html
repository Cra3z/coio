coio HTTP Server

This is a simple HTTP server built with coio, a C++20 async I/O library.

Features

  • C++20 coroutines with co_await
  • Async TCP/UDP networking
  • epoll/io_uring backend io context
  • Task-based concurrency
  • C++26 std::execution adaptation
